# Dotfiles

My personal configuration files for Arch Linux.

## Contents

- **Hyprland** - Wayland compositor configuration
- **Kitty** - Terminal emulator
- **Waybar** - Status bar
- **Rofi** - Application launcher
- **Neovim** - Text editor (submodule)
- **Flameshot** - Screenshot tool
- **SwayNC** - Notification center
- **GTK** - GTK theme settings

## Installation

Clone this repository to `~/.config`:

```bash
git clone --recurse-submodules <your-repo-url> ~/.config
```

## Neovim Configuration

The Neovim configuration is maintained as a separate repository and included as a git submodule.

## Notes

This configuration is tailored for my personal setup running Arch Linux with Hyprland. Adjust as needed for your environment.
